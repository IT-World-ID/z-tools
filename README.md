## INSTALL ON TERMUX
```
$ pkg up
$ pkg i ruby git ffmpeg tree zip unzip
$ git clone https://github.com/IT-World-ID/z-tools
$ cd z-tools
$ ruby main.rb
``` 

## INSTALL ON LINUX
```
$ sudo apt-get update
$ sudo apt-get install git ffmpeg ruby tree zip unzip
$ git clone https://github.com/IT-World-ID/z-tools
$ cd z-tools
$ ruby main.rb
```

## Supported Platform
+ Android [Termux]
+ Linux [Debian Family]

## Required 
+ Ruby
+ ffmpeg
